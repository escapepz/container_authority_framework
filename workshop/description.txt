[h1]Container Authority Framework[/h1]

[b]Core purpose:[/b]
A modding framework that lets creators build custom storage rules and container interactions.

[b]What it provides:[/b]
[list]
[*]Three-phase transfer pipeline - Validation (block transfers), Pre-Transfer (audit/modify), and Post-Transfer (side effects) hooks for container interactions
[*]Priority-based rule registration - Multiple mods can add rules without conflicting
[*]Sandbox performance controls - Server admins can configure listener limits to prevent lag
[*]Event-driven architecture - Rules trigger automatically when players move items between containers
[/list]

[b]Best for:[/b]
Building role-play, economy, and storage restriction systems for community servers.
